# lovely-farm
Project making CRUD of events using FARM stack

# Client:
- install: cd client && npm i
- run: npm run dev

# Server:
- install: cd server && python3 -m pip install -r requirements.txt
- run: python3 main.py 

# Swagger
- Go to http://localhost:8000/docs
